# MapleGPU 🇨🇦  
**Buy and Sell Used GPUs Across Canada**

MapleGPU is a full-stack web application that simplifies the process of finding and listing used graphics cards (GPUs) across Canada. Inspired by platforms like Jawa, this project is built with scalability, performance, and user safety in mind.

## 📦 Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/SameedBaqai94/maplegpu.git
   cd maplegpu
   ```

2. **Set up environment variables**
   - Copy `.env.example` to `.env` and update your database URL and other config as needed.

3. **Run with Docker**
   ```bash
   docker-compose up --build
   ```

4. **Access the app**
   - Backend API: `http://localhost:5000`
   - Frontend (if added to Compose): `http://localhost:3000`
